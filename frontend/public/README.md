# Public Assets

Place your logo and favicon images here:

1. **logo.png** - Main logo for the header (should have transparent background)
   - Recommended size: 40x40px to 80x80px
   - Format: PNG with transparent background
   - The image will be automatically resized to 40px height

2. **favicon.png** - Browser favicon
   - Recommended size: 32x32px or 64x64px
   - Format: PNG with transparent background
   - This appears in browser tabs

Both images should have transparent backgrounds to match the CyberMaze theme.
